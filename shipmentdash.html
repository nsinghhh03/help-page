Audit Cube: Shipment Dashboard

The Shipment Dashboard within Audit Cube serves as the main interface for users to view and analyze shipment-related data. Users can access shipment records, review statuses, and manage shipment-related disputes directly from this dashboard.

Shipment Table Overview

The Shipment Dashboard displays detailed shipment information, such as:

  • Shipment ID
  • Tracking Number
  • Shipment Date
  • Total Packages
  • Total Charges
  • Status

Shipment Details (Paper Icon)

The blue paper icon next to each shipment record opens the shipment details page, where users can view:

  • Shipper and recipient information
  • Carrier information
  • Service level
  • Shipment charges
  • Any disputes or discrepancies

Dispute Resolution

If a user identifies an issue with a shipment (e.g., incorrect charges, delayed delivery), they can raise a dispute directly from the dashboard. The dispute form allows users to provide a reason and upload supporting documents, if necessary.

Export Options

Users can export the shipment table to Excel for offline analysis. The export includes all shipment data fields, providing a comprehensive report for further processing.

Shipment Visualization

  • Shipment Volume Over Time: A chart visualizing the number of shipments processed over time, helping users understand trends and peak shipping periods.
  • Shipment Status Breakdown: A pie chart displaying the proportion of shipments in different statuses (e.g., Delivered, In-Transit, Delayed).

Troubleshooting

Shipment Records Not Appearing:

Issue: If shipment records are not appearing, it might be due to incorrect date filters or a technical glitch.

Solution:

  • Check Filters: Make sure your date filters or other filters are set correctly.
  • Refresh the Page: A simple page refresh can often solve loading issues.
  • Check Network Connection: Ensure your internet connection is stable.
