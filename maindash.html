Audit Cube - Main Dashboard

Table Overview

The main dashboard is organized in a table format that displays essential information such as Tracking Number, Shipment ID, Shipment Date, Total Packages, Total Shipment Charges, Tracking Status, Invoice Number, Invoice Date, and Invoice Amount.

Audit Results (Blue Paper Icon)

Located on the far left of the table, the blue paper icon takes users to the detailed audit results page for each shipment. This page provides a comprehensive breakdown of the shipment details, including shipper and recipient addresses, service codes, and other relevant data.

Advanced Search (Magnifying Glass Icon)

The search icon allows users to perform advanced searches across various parameters such as Tracking Number, Shipment ID, Invoice Number, Dispute ID, Shipment Date, and more.This feature enables users to quickly find specific records or filter the table to display only relevant data. You can also click a specific tracking or invoice number on the table to see its details.

Export to Excel

Positioned next to the search icon, the Excel icon allows users to export the table data into a Microsoft Excel spreadsheet. This is particularly useful for users who need to perform offline analysis or share data with other stakeholders.

Column Configurator

The Column Configurator tool enables users to customize their table view by hiding or displaying specific columns. This feature is valuable for users who need to focus on particular data points without the distraction of unnecessary information.

Visualization Tools & Insights

Freight Spend Per Month

A chart that summarizes the freight spend for the selected period, providing insights into monthly trends. Users can view their total freight spend and compare it across different months.

Audit Status Summary

Displays the status of invoices and shipments, showing the percentage of packages that have been audited and approved, are awaiting invoice, or have discrepancies. This visualization helps users quickly identify areas that need attention.

Auto Match Status

A doughnut chart that indicates the percentage of records that have been automatically matched, providing a quick overview of the audit accuracy and reducing manual workload.

Troubleshooting

Data Not Loading Properly

  • Issue: If the data within the dashboard fails to load, this might be due to a slow network connection or a temporary issue with the platform.
  • Solution:
    • Refresh the Page: Start by refreshing the page to see if the data loads correctly.
    • Check Network Connection: Ensure that your internet connection is stable.
    • Clear Browser Cache: Sometimes, cached data may cause loading issues. Clearing your browser’s cache can help resolve this.
    • Contact Support: If the issue persists, contact technical support for further assistance.

Advanced Search Not Returning Expected Results

  • Issue: When using the Advanced Search feature, you may not get the results you expected, which could be due to incorrect search parameters or filters.
  • Solution:
    • Double-Check Parameters: Ensure that all search parameters are entered correctly.
    • Clear Filters: If multiple filters are applied, clear them and reapply one at a time.
    • Broaden Search Criteria: Try broadening the search to capture more relevant data.

Export Issues (Excel Export Not Working)

  • Issue: Unable to export the table data to Excel.
  • Solution:
    • Try a Different Browser: Sometimes, certain browsers block pop-ups or downloads.
    • Check Pop-up Blockers: Ensure your browser’s pop-up blocker isn’t preventing the download.
    • Check File Size Limits: Large datasets might cause export issues. Try exporting smaller subsets of data.
